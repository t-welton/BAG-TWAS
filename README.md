# BAG-TWAS
Transcriptomic architecture of MRI-derived brain age gap.

These files are a project record, not a turnkey executable workflow.

Placeholders are used in both files to cover device names and paths.

* *bag-gloud-script.sh*: This is a bash script that was run on Google Cloud and performs the data harmonisation, imputation, and TWAS predictions
* *bag-r-script.r*: This r script was used for downstream analyses and figure generation including PCA, WGCNA, replication, causal inference (colocalisation, SMR+HEIDI), and conditioning
